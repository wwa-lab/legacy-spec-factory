# Calibration Scorecard: 9.0 Repo-Ready, Not Field-Ready

## Example Profile

A skill earns 9.0 when it is structurally strong and safe to merge, but still
needs hardening before internal field use.

## Common 9.0 Pattern

- `SKILL.md` is valid and portable.
- Workflow is complete.
- Output contract exists.
- SME review is present.
- Evidence discipline is present.
- Examples are shallow or missing.
- Runtime compatibility is designed but not tested.
- Negative cases are listed but not demonstrated.

## Decision

Repo-ready only. Do not use with real SME or engineering users until the missing
examples, runtime checks, and negative cases are addressed.
