# Calibration Scorecard: 8.0 Revise

## Example Profile

A skill is capped at 8.0 when it has promising content but fails one or more
mandatory controls.

## Common 8.0 Pattern

- Good narrative but weak `description`.
- No clear output file contract.
- Evidence tags are suggested but not required.
- SME review is mentioned only at the end.
- Runtime portability is assumed, not described.
- The skill encourages inferring business rules from incomplete evidence.

## Decision

Return to Claude Code for revision. Do not merge until the cap condition is
removed and the skill is rescored.
