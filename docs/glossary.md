# Glossary

| Term | Meaning |
| --- | --- |
| IBM i | IBM midrange operating system historically associated with AS/400 and iSeries |
| AS/400 | Earlier name for the IBM midrange platform; often used colloquially for IBM i systems |
| RPGLE | RPG IV / ILE RPG source used heavily on IBM i |
| CLLE | Control Language source used for job orchestration and command execution |
| ILE | Integrated Language Environment; IBM i model for modules, service programs, binding, and activation groups |
| OPM | Original Program Model; older IBM i program model before ILE |
| Service Program | ILE object containing reusable procedures that programs can bind to or call |
| Module | Compiled ILE object that can be bound into programs or service programs |
| DDS | Data Description Specifications used to define files, screens, and reports |
| PF | Physical File; table-like data object |
| LF | Logical File; keyed or joined access path over physical files |
| DSPF | Display File; screen UI definition |
| PRTF | Printer File; report/spool output definition |
| Spool File | Generated print/report output held in IBM i output queues |
| Subfile | DSPF pattern for scrollable lists in interactive screens |
| Indicator | RPG/DDS boolean flag often used for control flow, screen behavior, and conditioning |
| `SETLL` | RPG operation that positions a file cursor by key |
| `READE` | RPG operation that reads equal keys after positioning |
| Data Area | IBM i object used to store small shared values or control flags |
| Data Queue | IBM i object used for asynchronous message/data passing |
| Message Queue | IBM i object used for job, user, and application messages |
| Commitment Control | IBM i transaction control mechanism for commit/rollback behavior |
| Journal | IBM i object that records changes for recovery, audit, and commitment control |
| Journal Receiver | IBM i object that stores journal entries for a journal |
| Job Log | Runtime log for an IBM i job |
| Scheduler Entry | Job scheduler definition that controls when and how a job is submitted |
| CCSID | Coded Character Set Identifier; affects text encoding, export, and redaction handling |
| Control Table | Table or file that drives configurable business behavior |
| Golden Master | Legacy output used as an expected baseline for new-system comparison |
| SME | Subject-matter expert, especially IBM i or business process expert |
