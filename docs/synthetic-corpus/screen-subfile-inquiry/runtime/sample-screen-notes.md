# Sample Screen Notes

- User starts from `Customer Service Menu` and selects option `5`.
- Header screen shows customer name, status, and credit limit after customer
  number is entered.
- Subfile displays recent transactions with columns for date, type, amount, and
  status.
- `F5` clears and reloads the subfile.
- `F12` returns to the previous menu.
- Enter on the inquiry screen displays or refreshes the selected customer's
  transaction history.
