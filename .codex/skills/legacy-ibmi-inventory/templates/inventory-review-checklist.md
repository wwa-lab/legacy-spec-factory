# Inventory Review Checklist

- capability:
- reviewer:
- review_date:

## Coverage

- [ ] all expected RPGLE / COBOL programs are listed
- [ ] all expected CLLE programs and submitted jobs are listed
- [ ] all PF/LF objects are listed
- [ ] all DSPF objects are listed
- [ ] all PRTF and report outputs are listed
- [ ] spool samples or report examples are available where needed
- [ ] data areas, data queues, and message queues are listed or ruled out
- [ ] external interfaces are listed or ruled out
- [ ] copybooks and shared includes are listed or ruled out

## Evidence

- [ ] every object has at least one evidence ID or SME confirmation
- [ ] sensitivity is reviewed for every evidence item
- [ ] raw production data is not included in committed artifacts
- [ ] missing evidence is represented as a TBD

## Downstream Readiness

- [ ] inventory is sufficient for program analysis
- [ ] inventory is sufficient for DDS/schema analysis
- [ ] blocking TBDs are identified
- [ ] non-blocking TBDs are explicitly marked
