# Blocked Example: Missing Runtime Evidence

This example shows the required behavior when an approved acceptance criterion
has no redacted runtime evidence for its expected output.

The planner must not invent the missing rejection response. It emits a blocked
review and a machine-readable finding that routes remediation to evidence
collection or SME-approved deferral.
